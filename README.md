## 📖 專案簡介

本專案為**社團法人新北市道一關懷協會**的官方網站，提供協會資訊、文章分享及線上捐款功能。

---

## ✏️ 內容編輯指南

### 新增/編輯文章

編輯 `BLOG_CONSTANTS/_ARTICLES_LIST.tsx` 檔案來管理文章內容。

### 修改網站設定

編輯 `BLOG_CONSTANTS/_BLOG_SETUP.tsx` 來修改：

- 網站名稱與 Logo
- 導航列連結
- SEO 設定
- 社群連結

### 修改頁面內容

直接編輯 `pages/` 目錄下對應的 `.tsx` 檔案。
